== Description ==

Přeloží plugin shortcodes-ultimate do více jazyků včetně češtiny a slovenštiny.

Již dostupné překlady:

* Čeština
* Slovenština
* Ruština

== Installation ==

Instalace je velmi jednoduchá

1. Nainstalujte plugin přes administraci v Pluginy.
1. Activujte plugin.
